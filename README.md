# Portafolio personal

¡Bienvenido a mi portafolio personal!  
Este proyecto está desarrollado principalmente con **Vue.js**, junto a CSS y JavaScript, y tiene como objetivo mostrar mis proyectos, skills, certificaciones y forma de contacto de manera moderna y profesional.

## 📁 **Estructura básica**
```
/
├── public/         # Recursos estáticos y favicon
├── src/
│   ├── assets/     # Imágenes, certificados, iconos, etc.
│   ├── components/ # Navbar, tarjetas, sección de títulos, etc.
│   ├── views/      # AboutMe, Projects, Technologies, Certifications, ...
│   ├── App.vue
│   ├── main.js
├── package.json
├── vite.config.js || vue.config.js # (según el stack)
└── README.md
```

## 🛠️ **Requisitos previos**

- [Node.js](https://nodejs.org/) >= 16.x (recomendado)
- [npm](https://www.npmjs.com/) >= 8.x (se instala con Node.js)
> Alternativamente puedes usar yarn.

---

## ⚡ **Instalación y ejecución local**

1. **Clona este repositorio:**
   ```sh
   git clone https://github.com/Amonwill/Portafolio.git
   cd Portafolio
   ```

2. **Instala las dependencias:**
   ```sh
   npm install
   # o
   yarn install
   ```

3. **Inicia el servidor de desarrollo:**
   ```sh
   npm run dev
   # o
   yarn dev
   ```

4. Abre tu navegador y entra a  
   [http://localhost:5173](http://localhost:5173)  
   *(o el puerto que indique la terminal)*

---

## 🏗️ **Compilar para producción**

Genera la carpeta `dist` optimizada para desplegar en cualquier hosting estático (Netlify, Vercel, GitHub Pages, etc):

```sh
npm run build
# o
yarn build
```

Los archivos listos estarán en la carpeta `/dist`.

## 📝 **Comandos útiles**

| Comando            | Acción                         |
|--------------------|-------------------------------|
| `npm run dev`      | Servidor en modo desarrollo   |
| `npm run build`    | Genera build de producción    |
| `npm run preview`  | Prévia local de producción    |
| `npm install`      | Instala dependencias          |
